# ShotLog v0.11.1

- v0.7.0 계열 안정 녹화 구조 기준
- 영상 조각과 오디오 청크를 실제 시간 기준으로 맞춰 클립 저장
- 소리 동기 보정 0ms = 실제 0ms
- 출력 장치/마이크는 하나씩 선택
- PowerShell updater 적용

## v0.11.1
- 기본 Windows 제목 표시줄을 제거하고 ShotLog 전용 다크 상단바를 추가했습니다.
- 상단 중앙에 자동 녹화 상태 배지를 배치했습니다.
- 버튼/진행바 강조색을 아이콘의 민트색 계열로 통일했습니다.
- GitHub 소유자/저장소 입력칸을 제거하고 업데이트 저장소를 코드 내부 고정값으로 처리했습니다.
- 최근 로그 영역에 로그 출력 버튼을 추가했습니다.
- 저장 가능 영상 문구를 짧게 정리했습니다.
- 온글잎 베리려원 TTF를 프로그램 내장 폰트로 적용했습니다.

## v0.10.1
- 오디오 청크를 이벤트 시각마다 끊어 배치하지 않고, 캡처된 샘플 수 기준의 연속 타임라인으로 저장하도록 수정했습니다.
- 0.5초 간격으로 오디오가 툭툭 끊기는 현상을 줄였습니다.
- 소리 동기 보정 0ms는 실제 0ms 기준을 유지합니다.
